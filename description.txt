If Cheat Sheet is enabled, this mod will create a button for Cheat Sheet to turn the player invisibled when pressed.
If HERO's Mod is enabled and Cheat Sheet isn't enabled, this mod will create a button for HERO's Mod to turn the player invisibled when pressed.
If HERO's Mod and Cheat Sheet are not enabled, this mod will rely on this mod's config to turn the player invisibled.
